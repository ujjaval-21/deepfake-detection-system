## 🤖 Deepfake Detection System 👱

A web-based AI-powered system that detects whether an image and video is real or deepfake.

## Features 🌟
- 🖼️ Image Deepfake Detection
- 🎥 Video Deepfake Detection
- 🌐 Metadata Inspection
- ⏳ Detection History Tracking

## Tech Stack 🗂️
- **Backend:** Python Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Libraries:** OpenCV, Librosa, Numpy, Pillow

## How to Run
1. **📦📥 Install requirements**
- pip install -r requirements.txt

2. **🌐 Run server**
- python app/app.py

3. **🌐 Open browser**
- http://127.0.0.1:3002
